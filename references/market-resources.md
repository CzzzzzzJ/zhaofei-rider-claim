## 4. 客户承接系统

```
抖音视频曝光 → 评论区话术（见 douyin-templates.md）
   ↓
私信引导 → 加微信 → 发 H5 问卷（3 分钟填完）
   ↓
飞书多维表格自动归档 → 客服 Bot（见 prompts/customer-service.md）初判
   ↓
人工跟进 → 产品介绍 → 签"服务委托协议" → 拉群 → 服务跟进
   ↓
理赔到账 → 第三方银行卡自动扣服务费
```
